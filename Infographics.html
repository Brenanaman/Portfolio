Infographic Poster

Program » Adobe Illustrator

Format » 22in x 28in

Our assignment for this project was to employ information graphics to create a visual guide for comprehending a societal issue. I decided to compose a piece about the issues associated with cyberbullying. I needed some consistency, so I utilized largely pie charts, with my first one being different to catch attention. I also selected to use the colors of the awareness organizations for bullying and cyberbullying. The fact that this project explored visual communication with a data-driven design approach was another reason I liked it.
